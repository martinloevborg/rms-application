# Remote Monitoring System application

## Project structure

Backend – API
Frontend – Vue 3 web app with TypeScript

## How to run

### Frontend

1. Go inside folder

   ```bash
   cd frontend
   ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Run the project

   ```bash
    npm run dev
   ```

### Backend

1. Go inside folder

   ```bash
   cd backend/API
   ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Build and run

   ```bash
    npm start
   ```
